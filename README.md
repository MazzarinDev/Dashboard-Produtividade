# Dashboard de Produtividade

Este projeto consiste em um Dashboard de Produtividade desenvolvido em Python, utilizando o framework Streamlit, para rastreamento de tarefas, horas dedicadas e consumo de água. Foi criado como uma atividade de extensão e complementar para a faculdade, totalizando 70 horas de dedicação.

## Funcionalidades

*   **Rastreamento de Horas:** Simulação de 70 horas de trabalho distribuídas em categorias (Estudo, Extensão, Pessoal).
*   **Controle de Água:** Monitoramento do consumo diário de água com meta de 2000ml.
*   **Visualização de Dados:** Gráficos interativos (Plotly) para análise da distribuição de tempo e progresso de metas.
*   **Formulário de Demonstração:** Campo de entrada para simular o registro de novas atividades, horas de estudo e consumo de água.

## Tecnologias Utilizadas

*   **Python**
*   **Streamlit**
*   **Pandas**
*   **Plotly Express**
*   **NumPy**

## Como Executar

1.  **Clone o repositório:**
    ```bash
    git clone [LINK_DO_REPOSITORIO]
    cd dashboard-produtividade
    ```
2.  **Instale as dependências:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Execute o dashboard:**
    ```bash
    streamlit run app.py
    ```
